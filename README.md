## Overview

This repository contains the planning artifacts and autonomous agent configuration for building Seven Fortunas's AI-native enterprise infrastructure.

**Project Goal:** Create a comprehensive infrastructure consisting of:
1. GitHub Organizations (public + internal)
2. Second Brain (knowledge management system)
3. 7F Lens Intelligence Platform (multi-dimensional dashboards)
4. BMAD Library Deployment (70+ workflow skills)
5. User Profile System (AI agent personalization)
6. Voice Input System (cross-platform speech-to-text)

**Approach:** BMAD-first methodology + Claude Code SDK autonomous agent

**Timeline:** 5-day MVP (60-70% automated, 30-40% human refinement)

## Quick Start

### Prerequisites

1. **Python 3.10+** with `claude-agent-sdk`:
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   pip install claude-agent-sdk
   ```

2. **Anthropic API Key**:
   ```bash
   export ANTHROPIC_API_KEY="sk-ant-..."
   ```

3. **GitHub CLI authenticated**:
   ```bash
   gh auth login
   gh auth status
   ```

### Running the Autonomous Agent

**After PRD and `app_spec.txt` are complete:**

```bash
# Navigate to project directory
cd /home/ladmin/seven-fortunas-workspace/7f-infrastructure-project

# First run (initializer agent)
./scripts/run_autonomous.sh

# Continuous mode (auto-restarts until complete)
./scripts/run_autonomous_continuous.sh
```
