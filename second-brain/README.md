# Seven Fortunas Second Brain

Structured knowledge scaffold for the Seven Fortunas organization.
Mirrors the canonical [`seven-fortunas-brain`](https://github.com/Seven-Fortunas-Internal/seven-fortunas-brain) repository layout.

## Structure

| Directory | Purpose |
|-----------|---------|
| `best-practices/` | Engineering, AI, and operational standards |
| `brand/` | Brand guidelines, voice, and visual identity |
| `culture/` | Company values, ways of working, team norms |
| `domain-expertise/` | AI/ML knowledge, fintech, industry context |
| `operations/` | SOPs, runbooks, and operational playbooks |
| `skills/` | Team skills inventory and learning resources |

## Depth Constraint

Maximum 3 levels deep to keep navigation simple and search fast.
