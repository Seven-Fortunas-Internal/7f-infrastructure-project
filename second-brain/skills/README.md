# Skills

This domain contains documentation for BMAD (Behavioral Modeling and Design) skills and custom skills developed for Seven Fortunas.

## Navigation

- **bmad-skills/** — BMAD framework skills and modules
- **custom-skills/** — Organization-specific skills and extensions

## Purpose

The Skills domain catalogs all available skills and capabilities, enabling team members to discover, understand, and effectively utilize both standard BMAD tools and custom Seven Fortunas extensions.

## Key Documents

### BMAD Skills
Standard BMAD framework documentation:
- Workflow management and creation
- Task automation and orchestration
- Skill development and packaging
- Integration patterns and middleware
- BMAD module reference
- Best practices for skill composition

### Custom Skills
Seven Fortunas-specific capabilities:
- Skill catalog and descriptions
- Skill usage examples and patterns
- Custom workflow extensions
- Integration with external systems
- Skill development guidelines
- Testing and deployment procedures

## Subdomains

Skills are organized into functional categories:
- **workflow/**: Workflow creation and management tools
- **automation/**: Task and process automation
- **integration/**: External system integrations
- **utilities/**: Helper skills and utilities
- **custom/**: Organization-specific implementations
