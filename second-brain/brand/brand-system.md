# Brand System

## Color Palette

### Primary Colors
- **Primary Blue** (#0052CC): Main brand color for primary CTAs and branding
- **Secondary Green** (#28A745): Success states, positive actions, confirmations
- **Accent Orange** (#FF6B35): Highlights, important alerts, attention-drawing elements
- **Neutral Gray** (#6C757D): Secondary text, borders, disabled states

### Usage Guidelines
- Use Primary Blue for main navigation and CTAs
- Use Secondary Green for success messages and confirmation dialogs
- Use Accent Orange sparingly for critical alerts
- Use Neutral Gray for supporting elements and text

## Typography

### Font Families
- **Inter** (primary): Clean, modern sans-serif for all UI and body text
- **Fira Code** (monospace): For code blocks, terminal output, and technical content

### Type Scale
- **H1** (2.5rem): Page titles and major headings
- **H2** (2rem): Section headings
- **H3** (1.5rem): Subsection headings
- **Body** (1rem): Standard paragraph text
- **Small** (0.875rem): Secondary text, captions, metadata

## Logo Usage

### Logo Variations
- **Full Logo**: Organization name with icon (primary usage)
- **Icon Only**: Standalone symbol (favicons, app icons)
- **Horizontal**: Logo laid out horizontally (web headers)
- **Vertical**: Logo laid out vertically (print materials)

### Clear Space
Maintain minimum clear space of 1em around all logo variants.

### Background Requirements
- Logo should always have sufficient contrast
- Avoid placing on busy backgrounds
- Minimum size: 48px width for full logos, 24px for icon-only

## Spacing and Grid

### Grid System
- 8px baseline grid for consistency
- Spacing increments: 4px, 8px, 16px, 24px, 32px, 48px, 64px

### Component Spacing
- Padding: 16px (cards), 24px (sections)
- Margins: 16px (between sections), 8px (between elements)
- Gaps: 8px (inline elements), 16px (vertical stacks)

## Imagery Guidelines

### Photography
- Use authentic, high-quality photography
- Prefer diverse, inclusive imagery
- Maintain consistent color grading across assets

### Illustrations
- Use a consistent illustration style
- Ensure accessibility with alt text
- Maintain visual hierarchy

## Component Design

### Buttons
- Primary: Primary Blue background, white text
- Secondary: Gray background, dark text
- Outlined: Transparent with colored border
- Disabled: 50% opacity

### Cards
- 8px border radius
- Light shadow for depth
- 16px internal padding

### Badges and Tags
- 4px border radius
- Smaller typography (0.875rem)
- Appropriate color coding for category
