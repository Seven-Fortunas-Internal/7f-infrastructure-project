# Brand

This domain contains the brand identity, visual system, messaging guidelines, and organizational tone for Seven Fortunas.

## Navigation

- **brand.json** — Master brand configuration and metadata
- **brand-system.md** — Visual identity, color palette, typography, and design system
- **tone-of-voice.md** — Messaging guidelines, vocabulary, and communication standards

## Purpose

The Brand domain establishes consistent identity and communication across all Seven Fortunas touchpoints. This includes external-facing brand assets and internal communication standards.

## Key Documents

### brand.json
Master configuration file containing:
- Organization name, tagline, mission statement
- Brand colors (hex codes, RGB values)
- Typography definitions
- Asset URLs and locations
- Legal metadata (copyright, trademark info)

### brand-system.md
Visual design system documentation:
- Color palette and usage guidelines
- Typography scale and font recommendations
- Logo variations and usage rules
- Spacing and grid systems
- Imagery and photography guidelines
- Component design patterns

### tone-of-voice.md
Communication standards:
- Brand voice personality traits
- Vocabulary and terminology glossary
- Writing style guidelines
- Email and messaging templates
- Social media voice guidelines
