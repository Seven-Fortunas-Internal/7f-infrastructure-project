### Method 1: Browse (2 clicks)

1. **Click:** Navigate to domain section above
2. **Click:** Select specific document

**Example:** Find access control docs

---

### Method 2: grep Search (15 seconds)

```bash
# Search for keyword in all docs
cd docs/
grep -r "keyword" .

# Search for specific pattern
grep -r "vulnerability" . | grep -i "sla"

# Search in specific subdirectory
grep -r "2FA" security/

# Case-insensitive search
grep -ri "github app" .
```

**Example:** Find documentation about secret detection  
```bash
cd docs/
grep -ri "secret detection" .
```

---

### Method 3: GitHub Search (15 seconds)

1. Press `/` (GitHub search shortcut)
2. Type keyword
3. Filter by path: `path:docs/`

**Example:** Find vulnerability documentation  
Search: `vulnerability path:docs/`

---

### Method 4: AI-Assisted Search (< 2 minutes)

Use natural language queries referencing YAML frontmatter:

**Example Queries:**
- "Show me all SOC 2 documentation"
- "Find architecture docs updated in the last week"
- "Where are the access control audit procedures?"

**AI reads YAML frontmatter:**
```yaml

category: Security
```
