# Templates

## Purpose

This directory contains templates for code generation, scaffolding, and standardization across Seven Fortunas projects.

## Contents

- **second-brain-frontmatter-template.md** - YAML frontmatter template for Second Brain markdown documents (FR-2.2 compliant)

## Usage

Templates are used by:
- Skills (7f-skill-creator, 7f-sop-generator, etc.)
- Scripts (generate-readme-tree.sh, etc.)
- Manual project setup and documentation

### Using Templates

1. Copy template file to target location
2. Replace placeholder values
3. Customize as needed for specific use case

### Adding New Templates

When adding new templates:
1. Follow naming convention: `<purpose>-template.<ext>`
2. Include comments/placeholders for required customization
3. Document template usage in this README
4. Reference template in relevant skills/scripts
