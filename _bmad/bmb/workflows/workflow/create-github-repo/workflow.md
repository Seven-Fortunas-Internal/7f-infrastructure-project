# Create GitHub Repo

**Goal:** Set up production-ready GitHub repositories with security, collaboration, and automation features.

## Steps

1. **Define Repository**: Specify name, description, public/private status
2. **Configure Basics**: Set default branch, add README and license
3. **Enable Features**: Configure issues, discussions, projects, wiki as needed
4. **Set Permissions**: Configure team access, branch permissions
5. **Add Protection**: Set up branch protection rules, required reviews
6. **Enable Security**: Configure secret scanning, dependabot alerts
7. **Setup Automation**: Add webhooks, GitHub Actions, integrations
8. **Verify Configuration**: Validate all settings are correct

## Tools

- GitHub CLI (gh) for repository operations
- GitHub API for advanced configuration
- Terraform/IaC for infrastructure-as-code repos
- Security checklist templates
