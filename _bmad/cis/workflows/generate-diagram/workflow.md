# Generate Diagram

**Goal:** Produce diagrams that clarify complex concepts and relationships visually.

## Steps

1. **Understand Subject**: Clarify what needs to be visualized
2. **Choose Diagram Type**: Select appropriate visualization (flow, network, timeline, etc.)
3. **Identify Elements**: List components, entities, and relationships
4. **Sketch Layout**: Plan spatial arrangement and flow
5. **Create Diagram**: Build diagram using appropriate tools
6. **Add Labels**: Clearly label all elements and connections
7. **Verify Accuracy**: Check that diagram accurately represents the concept
8. **Optimize Visual**: Refine colors, spacing, and visual balance

## Tools

- Diagram editors (Excalidraw, Lucidchart, Draw.io, etc.)
- Visualization libraries (Graphviz, Mermaid, etc.)
- Flowchart templates
- Icon and shape libraries
- Color and accessibility checkers
