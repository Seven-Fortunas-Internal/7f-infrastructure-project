# Generate PPTX

**Goal:** Create visually appealing, persuasive presentations that communicate effectively.

## Steps

1. **Define Objectives**: Clarify presentation goal, audience, and key messages
2. **Outline Structure**: Create slide-by-slide outline and talking points
3. **Gather Assets**: Collect images, data, and supporting materials
4. **Design Slides**: Create visually consistent, uncluttered slides
5. **Add Messaging**: Write clear, concise slide text
6. **Include Visuals**: Add charts, images, diagrams for clarity
7. **Add Speaker Notes**: Document presentation flow and details
8. **Review & Polish**: Check consistency, flow, and impact

## Tools

- PPTX generation libraries (python-pptx, etc.)
- Design templates and themes
- Chart and visualization tools
- Image and asset management
- Presentation review tools
