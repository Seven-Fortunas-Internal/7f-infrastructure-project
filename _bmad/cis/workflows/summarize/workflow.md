# Summarize

**Goal:** Distill complex information into clear, actionable summaries.

## Steps

1. **Read/Analyze Source**: Thoroughly understand the content to summarize
2. **Identify Key Points**: Extract the most important information
3. **Determine Length**: Decide summary length (one-liner, paragraph, bullet points)
4. **Write Summary**: Compose concise summary maintaining accuracy
5. **Verify Completeness**: Ensure key information is preserved
6. **Simplify Language**: Use clear language appropriate for audience
7. **Add Context**: Provide brief context if needed for understanding
8. **Review**: Check accuracy and clarity of summary

## Tools

- Text summarization templates
- Key point extraction techniques
- Note-taking systems
- Content analysis tools
- Summary formatting guidelines
