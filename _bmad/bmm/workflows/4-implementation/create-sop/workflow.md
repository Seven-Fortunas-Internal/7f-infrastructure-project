# Create SOP

**Goal:** Document repeatable processes with step-by-step instructions and quality standards.

## Steps

1. **Identify Process**: Define the operational procedure to document
2. **Map Current State**: Document how it's currently performed
3. **Optimize Flow**: Identify improvements and efficiency gains
4. **Write Instructions**: Create clear, numbered steps with examples
5. **Add Decision Points**: Include when/how to handle exceptions
6. **Define Success**: Specify completion criteria and quality gates
7. **Review & Validate**: Get stakeholder sign-off

## Tools

- Process documentation templates
- Decision tree builders
- Approval workflow systems
- Version control for SOPs
